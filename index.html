Git Reset Command

Git Reset is used to move the current branch to a previous commit. It can also unstage files or completely remove changes depending on the option used.

Soft Reset

git reset --soft HEAD~1

Removes the last commit but keeps all changes staged.

Mixed Reset (Default)

git reset HEAD~1

Removes the last commit and unstages the changes.

Hard Reset

alok choudhary

git reset --hard HEAD~1

Removes the last commit and permanently deletes all changes.

Warning

Avoid using --hard unless you're sure. Deleted
